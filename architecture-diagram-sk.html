Kael — architektúra systému

Osobný AI asistent postavený nad Claude Code · prehľad pre tata · 2026-04-19

Ako to funguje

  • • Miro píše správu na Discord (chat).
  • • Discord plugin doručí správu do Kael CLI — hlavnej terminálovej session bežiacej na Macu.
  • • Claude spracuje s plným prístupom k auto-pamäti, vault-u (cez Smart Connections) a nástrojom.
  • • Odpoveď pošle späť na Discord.
  • • Dreamer agent každú hodinu zapíše poznatky z rozhovoru do vault-u; GitHub ho zálohuje každých 30 minút.

Ako to spustiť

  • • claude — hlavná Kael session v termináli
  • • ~/bin/run-agents dreamer — manuálny dreamer
  • • ~/bin/run-agents deep-dreamer — údržba vault-u
  • • launchctl load ~/Library/LaunchAgents/com.*.plist — plán agenta
  • • crontab -e — len shell úlohy (napr. auto-commit)
  • • obsidian ~/KaelVault — otvorí vault v Obsidian app
  • • gh repo clone MaoTanx/KaelVault — obnova zo zálohy

Kľúčové fakty

  • • 1 aktívny hook (discord-reply-check)
  • • 1 cron (auto-commit) + 2 launchd (dreamer, deep-dreamer)
  • • ~35 súborov v auto-pamäti
  • • stovky poznámok v KaelVault-e
  • • GitHub zálohuje každých 30 minút
  • • Claude Opus 4.7 cez Max-plán (bez per-token platenia)

Glosár — čo každá komponenta robí

Jedna veta na každú vec. Farebný štítok = zdroj (viď legenda v diagrame).

Claude Opus 4.7 Anthropic
Veľký jazykový model (AI mozog), ktorý spracováva každú správu — rozumie, odpovedá, volá nástroje; beží v cloude Anthropic-u a prístup máme cez Max-plán predplatné.
Claude Code CLI (claude) Anthropic
Terminálová aplikácia, ktorá hostí session s Claudom a dáva mu prístup k nástrojom (Bash, Read/Write, Git, MCP servery); v tomto setup-e je to „telo" Kaela.
Superpowers plugin Anthropic
Oficiálny plugin z Anthropic marketplace-u, ktorý pridáva 14 pracovných „skill-ov" — brainstorming pred tvorbou, writing-plans, TDD, dispatching-parallel-agents a ďalšie.
claude-agent-sdk Anthropic
Python knižnica od Anthropic-u, ktorá umožňuje spúšťať Claude agentov z vlastného kódu — používa ju napr. skript, ktorý každú hodinu fire-uje dreamer.
Discord (chat kanál) Discord
Chatovacia platforma — primárny spôsob komunikácie Miroa s Kaelom (z mobilu aj PC); v kanáli je bot „MaoTanx", ktorý správy doručuje do Kael CLI.
plugin:discord:discord (MCP) Discord
Claude Code plugin, ktorý prepája Claudovu session s Discord API — Claude dostáva správy, posiela odpovede, vie čítať prílohy a reagovať emoji.
CLAUDE.md — hlavný súbor Vlastné
Jadro celého setup-u. Markdown súbor v ~/CLAUDE.md s permanentnými pravidlami — identita Kaela, trust hierarchia pre Discord, komunikačné pravidlá, bezpečnostné zámky (email, TV, súbory), inštrukcie pre prácu s pamäťou. Claude Code ho automaticky načíta na začiatku každej session a prežije aj kompakciu kontextu.
Auto-pamäť Vlastné
Priečinok ~35 malých markdown súborov (v ~/.claude/...), ktoré Claude automaticky načíta na začiatku každej session — osobnosť, pravidlá, profily ľudí, preferencie.
Dreamer (agent) Vlastné
Script, ktorý každú hodinu prečíta nové správy z transkriptu rozhovoru a pretaví ich do štruktúrovaných zápiskov v KaelVault-e — chat sa mení na dlhodobú pamäť.
Deep-dreamer (agent) Vlastné
Agent, ktorý raz denne (o 03:30) upratuje vault — zlučuje duplikáty, opravuje pokazené wiki-odkazy, čistí MEMORY.md index a pridáva krížové referencie.
Ďalší agenti (research-explorer) Vlastné
Špecializovaní sub-agenti, ktorých si Claude zavolá keď treba hlboký prieskum kódu / webu / dokumentácie; každý má vlastný systémový prompt a obmedzené nástroje.
Vlastné skilly Vlastné
Opakovateľné pracovné postupy napísané ako markdown — ultrareview-local dispatch-ne paralelných reviewer-ov; architecture-diagram vygeneroval tento diagram.
Hook: discord_reply_check Vlastné
Python script, ktorý sa spustí na konci každého turnu — skontroluje, či Claude odpovedal na Discord; ak nie, zablokuje koniec turnu kým odpoveď nenapíše.
auto-commit.sh Vlastné
Bash script, ktorý každých 30 min skopíruje auto-pamäť do vault-u, spraví git commit a push na GitHub — pred tým prebehne secret-scan proti únikom tokenov.
run-agents Vlastné
Python wrapper, ktorý cron-y používajú na spustenie dreamer / deep-dreamer — vyberie správne session-y na spracovanie a odovzdá SDK potrebné povolenia.
kael-toolkit Vlastné
Zdieľané Python prostredie (~/tools/kael-toolkit/), kde býva claude-agent-sdk a vlastné pomocné moduly — projekty sa naň odkazujú namiesto inštalácie SDK zakaždým.
KaelVault Obsidian
Priečinok markdown súborov organizovaných ako znalostná báza (Daily/, People/, Projects/, System/) — Miro ho prezerá v Obsidian app, dreamer doň píše, GitHub zálohuje.
Obsidian aplikácia Obsidian
Desktop app, v ktorej Miro prezerá a edituje KaelVault ako súvislú wiki s graph-view, wiki-odkazmi a search-om; zdarma, beží lokálne, bez cloud závislosti.
Smart Connections MCP Obsidian
Plugin do Obsidian-u, ktorý sa zároveň tvári ako MCP server — poskytuje Claudovi nástroje na sémantické hľadanie v KaelVault-e (AI embeddings, nie iba fulltext).
GitHub (zálohovanie) GitHub
Cloudové úložisko git repozitárov — každých 30 min sa tam pushne KaelVault a projekty; funguje ako off-site backup. Strata Mac-u = žiadna strata dát.
gh CLI GitHub
Oficiálny GitHub príkazový riadok — Claude ho používa cez Bash na vytváranie pull request-ov, archiváciu repozitárov, read/write issue; auth cez macOS keychain.
Gmail SMTP + IMAP Tretie strany
Prístup k Miroovmu gmail účtu (maotanx@gmail.com) cez SMTP na odosielanie a IMAP na čítanie — Claude posiela emaily keď Miro požiada; poverenia v .env súbore.
Anthropic API Tretie strany
Vzdialený endpoint Claude LLM — volania idú cez Max-plán OAuth (nie cez per-token API key), takže sa platí paušál a volania sa rátajú do 5-hodinového limitu.
OpenAI Whisper Tretie strany
API od OpenAI na prepis hlasu na text — Claude ho volá keď príde voice message cez Discord, aby rozumel čo Miro hovoril.
Node.js + Bun Nástroje
JavaScript runtime-y — Node.js (v22) beží Discord plugin a niektoré skripty; Bun je rýchlejšia alternatíva používaná pre MCP servery.
Python 3.11 + uv Nástroje
Python runtime a uv (správca balíkov a virtuálnych prostredí, rýchlejšia náhrada pip+venv) — používané pre všetky Python skripty (dreamer wrapper, hook-y, …).
Playwright + Chrome Nástroje
Automatizácia webového prehliadača (stealth Chrome) — Claude ju používa keď treba niečo spraviť na webe cez UI (napr. generovanie hudby na Suno), nie cez API.
Tailscale Nástroje
Súkromná VPN sieť (zero-config), ktorá spája mobil, laptop a Mac Mini — umožňuje SSH / VNC z mobilu kdekoľvek vo svete bez otvárania portov v routri.
